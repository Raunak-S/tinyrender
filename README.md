# tinyrender

tinyrender is a Rust port of the C++ tinyrenderer project: https://github.com/ssloy/tinyrenderer

## Installation

```bash
git clone https://github.com/Raunak-S/tinyrender.git
```

## Usage

```bash
cargo run -r -- --obj-path <path_to_file>
```
